== Welcome to Glassroom

This is the Google TV based youtube viewer for classrooms. Made for teachers who use videos in class, but want to receive feedback from students. Glassroom provides a teacher mode and student mode. The teacher mode allows the teacher to set up youtube playlists, show videos, and display student feedback. The teacher mode is meant to be used on the Google TV. The student mode has buttons for the students to anonymously provide the teacher feedback about the video content. The student's view is a responsive web app that can be accessed via cell phones, tablets, and laptops.
